🧠 Semantic Layer for Enterprise Analytics & AI Integration

Master's Thesis
Hochschule Harz & Linde Engineering
Feb 2026 – Aug 2026

Business Challenge

Enterprise data models are locked behind SQL expertise, making them inaccessible to business users. There was no governed, natural-language interface for exploring and querying the data model, and no automated way to move from high-level data exploration into raw data analytics within a single workflow.

Solution

Phase 1 built a GenAI application connecting SAP LeanIX and Azure Databricks via an MCP server, enabling business users to explore the data model in natural language. Enterprise Solution

Apr 2026 – May 2026

Business Challenge

Solution

🤖 Power BI Modeling Assistant

Developer Tool
Linde Engineering
Feb 2026 – May 2026

Business Challenge

BI developers had no way to correct DAX syntax using AI without leaving the Power BI environment and relying on external tools outside the company's infrastructure. At the same time, auditing a .pbix file against the internal Power BI SOP was a fully manual process, making it slow and inconsistent to catch violations such as incorrect column naming, missing measure tables, or measures placed in the wrong tables.

Solution

Extended Microsoft's open-source Power BI Modeling MCP Server into a complete installable application that auto-detects any open .pbix file and connects it to a Databricks-hosted AI model via token-based authentication, keeping all interactions within the company's infrastructure. The internal Power BI SOP was embedded directly into the assistant, making every interaction SOP-aware. Developers can ask questions about their semantic model and the assistant proactively flags non-compliances such as naming convention violations, missing measure tables, and misplaced measures, and can apply the fixes directly to the model on the developer's behalf.

Tech Stack
Python Power BI Databricks AI MCP GenAI
